##Запуск проекта

1. Открыть solution в Visual Studio 2022
2. Нажать Run
3. Открыть Swagger
4. Архитектура

Проект построен как конвейер обработки HTTP запросов.

Каждый middleware выполняет одну функцию:

LoggingMiddleware — логирует запрос TimingMiddleware — измеряет время ExceptionMiddleware — переводит исключения в JSON ошибки

Контроллеры содержат только бизнес-логику.

Хранилище реализовано в памяти через singleton сервис.

Такой подход делает систему расширяемой и понятной.
